About Me

I am a Web Developer with hands-on experience in building responsive and user-friendly websites using modern technologies. I specialize in front-end development while also having experience working with back-end integrations and CMS platforms.


I have worked with tools and technologies including HTML, CSS, JavaScript,PHP, React, Webflow,HubSpot and WordPress (including Elementor and Divi Theme Builders) to create clean, functional, and visually appealing websites. I focus on mobile-first design and ensuring a smooth user experience across all devices.


I use tools like Chrome DevTools, Lighthouse, and Google PageSpeed Insights to inspect, debug, and optimize websites. These tools help me improve performance, accessibility, and SEO, while ensuring strong Core Web Vitals such as loading speed, interactivity, and visual stability.


I have experience in WordPress custom theme development, where I build fully customized themes. I work with WordPress core functions and template structures to create dynamic and scalable websites tailored to client requirements.


My work includes website development, customization, bug fixing, and performance improvements. I also have experience with API integrations, as well as working with cloud platforms like AWS for deployment and hosting. I focus on optimizing websites for speed, scalability, and SEO best practices to deliver high-quality user experiences.


I am continuously learning and improving my skills, with a strong interest in modern web development tools and frameworks to build better and more efficient digital solutions.
